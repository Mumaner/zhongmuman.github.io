Markdown使用

Author: 木蛮 Date: 四月 9, 2022  19:13:33 Category: 乱七八糟 blog

markdwon使用

使用情景：blog

准备工作

  • 我的软件：Visual Studio Code
    markdown预览：ctrl+shift+p - 搜索markdown:open locked preview to the side

    • 有人推荐typora，但是我下了之后没找到预览视图，用不习惯。

  • markdown抬头格式

常见元素

元素 对应符号 描述
标题 #（个数越多级数越多） 逻辑：markdown（ Heading level 1）–html（<’h1>Heading level 1</‘h1>）–渲染输出
段落 空白行
换行 在该行末尾打两个以上空格后enter 或者直接分段吧[i.e.空白行]
着重 *或_ 一个为斜体两个为加粗三个为又斜又粗
删除线 ~
清单 1.（有序清单）[下一级：缩进即可]；-或+或*(符点无序清单) 数字不必按数字顺序排列，但列表应以数字开头。（即使你的数字编码是乱的，它也会以正确顺序输出）； 要在保留列表连续性的同时在列表中添加另一个元素，则需要缩进
水平线 ***或—或____ 三个以上即可
  • 注意事项：
    • 使用空格
    • 四个空格或tab为缩进
    • 中英文符号
    • 可以多用清单符号创建层级

不常见使用

  • 表格
    • 基本格式
      Syntax Description
      Header Title
      Paragraph Text
    • 示例
      标题 内容
      名字 木蛮

      -

    • 对齐
      ：[左对齐；居中；右对齐]

- 任务清单
- []空白方块； - [x] 打勾的方块
注：vs code没有此功能（可能需要安装拓展吧）。我没有学会如何安拓展（我觉得是得用梯子）

  • 链接到标题【标题编号】
    示例：
    `### My Great Heading
